Comandos basico do git

git init - Inicializa um repositorio git no diretorio atual.
git add * - Adiciona todos os arquivos, pastas e subpastas que se encontram no diretorio.
git add <arquivo> - Adiciona um unico arquivo ao repositorio.
git commit -m "Mensagem" - Salva as alterações feitas no repositorio.
git push - Envia as alterações para um repositorio remoto previamente configurado.
